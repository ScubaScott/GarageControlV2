pure virtual

Restores all live subsystem values from NV members.

Implements Rule 5 (NV → live) for the LoadNV menu screen. Does NOT re-read EEPROM; uses the in-RAM NV member variables.

◆ notifyLiveValueChanged()

virtual void IMenuHost::notifyLiveValueChanged ( )
pure virtual

Notifies that a live subsystem value has been modified.

Called by menu and MQTT handlers when a live value (not NV) is changed. Starts the 24-hour auto-revert timer. If 24 hours pass without another change, live values revert to their NV counterparts.

Note
NOT called during LoadNV() or initialization.
