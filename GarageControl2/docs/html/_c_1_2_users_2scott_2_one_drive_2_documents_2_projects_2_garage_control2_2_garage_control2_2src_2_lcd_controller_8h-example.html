Marks the display as dirty to force a complete refresh.

Marks the display as dirty to force a complete refresh.Used when the system state changes significantly and the display should be redrawn completely on the next updateDisplay() call. Normal polling updates are optimized to only refresh changed portions; this forces full-screen redraw.

Parameters
WakeBackLightIf true, also turns on the backlight and resets the backlight timeout. Use when user activity occurs.

// After door state change from external MQTT command: lcd.SetDirty(true); // Force redraw and wake the backlight
